## NLW eSports Web

Encontre o seu duo e se divirta em seus jogos favoritos!!!

## Tecnologias

Aqui estão algumas das tecnologias usadas neste projeto.

- ReactJS
- Radix-ui
- Axios
- Tailwindcss
- React Router Dom
- Typescript
- Phosphor Icons
- Vite
- Swiper
